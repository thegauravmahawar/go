## Commands

**Run a Go program**

```shell
go run 01-hello-world.go
```

**Build a Go program**

Sometimes we'll want to build our programs into binaries. We can do this using `build` option.

We can then execute the binary directly.

```shell
go build 01-hello-world.go

./01-hello-world
```
